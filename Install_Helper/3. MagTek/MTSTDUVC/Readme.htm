MagTek Standard (VCP) Driver 2K/XP/03/Vista/Win7

To install this driver, follow these steps:

Preinstall (Win 2K/03/XP/Vista/Win7):

  1. Run the executable install.bat from the appropriate path 32bit or 64bit. This step need only be performed once to cover all supported devices.

    For Vista and Win7:
    Run a command prompt with administrator privilages then type install.bat from the appropriate path 32bit or 64bit.
  2. Attach the device. Windows will indicate a successful install.

Manual Install:

  1. Attach the device. Windows will indicate that it found new hardware and will show IntelliStripe 65, IntelliStripe 320, IntelliStripe 380, IntelliPIN SPT, or MagneSafe MiniMICR device has been attached.
  2. You will then be prompted to use the USB Wizard to install the device driver and other appropriate files.
  3. When prompted, ask the Wizard to search for a suitable device driver from a specific location.
  4. Click the Browse button to specify the location. If you have the MagTek USB drivers on a CD, specify the CD drive and appropriate path (32bit, 64bit). If you used the web installation, specify the location to where the files were extracted.
  5. After you locate the requested INF file, click Open. 
  6. After all of the files have been installed, click Finish.

After the files have been installed, any application program can communicate with the IntelliStripe 65, IntelliStripe 320, IntelliStripe 380, IntelliPIN SPT, or MagneSafe MiniMICR just as if it is attached to a regular RS-232 COM port. If your application can automatically detect the available COM ports, the newly installed USB device will be shown as one of the available COM ports (e.g., COM5). If your application does not support COM port selection, you can determine the COM port number by using the device manager. This can be done by right-clicking on the My Computer icon on the desktop; then select Properties. In Windows 2000/2003/XP, click the Hardware tab, then Device Manager. In Windows Vista/Win7, click the Device Manger task. When the Device Manager window opens, click on the plus sign next to Ports (COM & LPT). The new device will be shown in the list with its COM port identified.
